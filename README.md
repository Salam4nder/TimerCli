# CLI Timer

> Simple timer with no dependencies at the time of writing (2022-06-26)

## Usage

```zsh 
timer min -v 5
```



```zsh 
timer sec -v 25
```

>Former sets a timer for 5 minutes, latter for 25 seconds.
